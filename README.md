# Student Grade Management System (C Language)

## Project Introduction
This project is created to practice version control using Git and GitHub.  
It is an individual assignment focusing on tracking code changes using commits.

## Project Description
This program is a Student Grade Management System written in C.  
It allows users to:
- Add student data
- Display all students
- Search student by ID
- Calculate average score
- Find highest score
- Save data to file
- Load data from file

## Version Control Scope
The following files are tracked:
- main.c
- student.c
- student.h
- README.md
- data.txt
